EV Assembly Line Optimization

Minimize idle time and bottlenecks in electric vehicle production.

Algorithm: Topological Sort (Kahn’s Algorithm)
Efficiency: O(V + E) 
Pseudocode:
1. Compute in-degree of tasks
2. Push tasks with in-degree 0 into queue
3. While queue not empty:
   Pop task, add to result
   Reduce in-degree of neighbors

C++:
vector topologicalSort(int V, vector>& adj) {
  vector in_degree(V, 0);
  for (int u = 0; u < V; u++)
    for (int v : adj[u]) in_degree[v]++;
  queue q;
  for (int i = 0; i < V; i++)
    if (in_degree[i] == 0) q.push(i);
  vector result;
  while (!q.empty()) {
    int u = q.front(); q.pop();
    result.push_back(u);
    for (int v : adj[u])
      if (--in_degree[v] == 0) q.push(v);
  }
  return result;
}

Textile Dyeing Slot Scheduling

Assign dyeing slots to textile batches to minimize overlap and energy usage.

Algorithm: Interval Scheduling (Greedy)
Efficiency: O(n log n) 
Pseudocode:
1. Sort jobs by finish time
2. For each job:
   If slot free, assign

C++:
struct Job { int start, end; };
bool compare(Job a, Job b) { return a.end < b.end; }
vector schedule(vector& jobs) {
  sort(jobs.begin(), jobs.end(), compare);
  vector result;
  int lastEnd = -1;
  for (Job j : jobs) {
    if (j.start >= lastEnd) {
      result.push_back(j);
      lastEnd = j.end;
    }
  }
  return result;
}
