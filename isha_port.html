Crop Yield Prediction

Case Study:
Can farmers predict harvest yields using soil nutrients, rainfall, and temperature data?

Algorithm: Multiple Linear Regression
Efficiency: O(n) per iteration

C++ Pseudocode:
double sigmoid(double x) { return 1.0 / (1.0 + exp(-x)); }

double predictYield(vector& features, vector& weights) {
    double result = 0;
    for (int i = 0; i < features.size(); i++) {
        result += features[i] * weights[i];
    }
    return result;
}

void trainRegression(vector>& X, vector& y,
                     vector& weights, double lr, int epochs) {
    for (int e = 0; e < epochs; e++) {
        for (int i = 0; i < X.size(); i++) {
            double pred = predictYield(X[i], weights);
            double error = y[i] - pred;
            for (int j = 0; j < weights.size(); j++) {
                weights[j] += lr * error * X[i][j];
            }
        }
    }
}

Conclusion:
Regression equips farmers with accurate forecasts, guiding planting and resource allocation decisions.

Irrigation Scheduling

Case Study:
How can limited water be distributed across fields to maximize crop health?

Algorithm: Dynamic Programming ‚Äì Knapsack
Efficiency: O(nW)

C++ Pseudocode:
int knapsack(int W, vector& water, vector& benefit, int n) {
    vector> dp(n+1, vector(W+1, 0));
    for (int i = 1; i <= n; i++) {
        for (int w = 0; w <= W; w++) {
            if (water[i-1] <= w) {
                dp[i][w] = max(dp[i-1][w],
                               dp[i-1][w-water[i-1]] + benefit[i-1]);
            } else {
                dp[i][w] = dp[i-1][w];
            }
        }
    }
    return dp[n][W];
}

Conclusion:
Dynamic programming ensures water is allocated optimally, boosting crop yield under scarcity.

Waste Management Routing

Case Study:
How can garbage trucks in Vishalnagari cover all collection points efficiently, saving fuel and time?

Algorithm: Travelling Salesman Problem (TSP) ‚Äì Nearest Neighbour + 2‚Äëopt heuristic
Efficiency: Approx O(n¬≤)

C++ Pseudocode:
vector tspNearestNeighbour(int n, vector>& dist) {
    vector visited(n, false);
    vector path;
    path.push_back(0); // start at depot
    visited[0] = true;

    for (int step = 1; step < n; step++) {
        int last = path.back();
        int next = -1, best = INT_MAX;
        for (int j = 0; j < n; j++) {
            if (!visited[j] && dist[last][j] < best) {
                best = dist[last][j];
                next = j;
            }
        }
        path.push_back(next);
        visited[next] = true;
    }
    // optional: apply 2-opt refinement to improve route
    return path;
}

Conclusion:
Heuristic TSP routing ensures trucks minimize travel distance, cutting costs and improving waste collection efficiency.

Soil Quality Classification

Case Study:
Can soil samples be classified into fertile or infertile categories to guide crop selection?

Algorithm: Decision Tree Classification
Efficiency: O(n log n) training, O(log n) prediction

C++ Pseudocode:
struct Node {
    string feature;
    string label;
    Node* left;
    Node* right;
};

string classify(Node* root, map& sample) {
    while (root->left && root->right) {
        if (sample[root->feature] == "yes") {
            root = root->left;
        } else {
            root = root->right;
        }
    }
    return root->label;
}

Conclusion:
Decision trees help farmers quickly classify soil, ensuring the right crops are planted for maximum yield.
