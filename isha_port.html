Efficiency: O(nW)

Pseudocode:

2. Initialize DP table with dimensions [fields][water capacity]

Soil Quality Classification

üîç Problem: Soil samples must be classified (e.g., fertile, sandy, clay) to recommend suitable crops.

Algorithm: Decision Tree Classification
Efficiency: O(n log n)

Pseudocode:
1. Collect soil dataset with features (pH, moisture, nutrients)
2. Choose best feature to split using information gain
3. Partition dataset into subsets
4. Recursively build decision tree until stopping criteria
5. Store leaf nodes with soil class labels
6. For new sample ‚Üí traverse tree based on feature values
7. Output predicted soil class
