Traffic Flow Optimization

Case Study:
Congestion clogs Vishalnagari’s busiest roads. How can vehicles be guided to minimize delays? Can shortest paths be computed dynamically?

Algorithm: Dijkstra’s Shortest Path
Efficiency: O(E + V log V)

C++ Pseudocode:
vector dijkstra(int V, vector>>& adj, int src) {
    vector dist(V, INT_MAX);
    priority_queue, vector>, greater<>> pq;
    dist[src] = 0; pq.push({0, src});
    while (!pq.empty()) {
        int u = pq.top().second; pq.pop();
        for (auto [v, wt] : adj[u]) {
            if (dist[u] + wt < dist[v]) {
                dist[v] = dist[u] + wt;
                pq.push({dist[v], v});
            }
        }
    }
    return dist;
}

Conclusion:
Dijkstra’s algorithm ensures smoother traffic by routing vehicles along the fastest available paths.

Signal Scheduling

Case Study:
Long waits at intersections frustrate drivers. How can signals be scheduled fairly across lanes? Can we rotate green lights to balance flow?

Algorithm: Round Robin Scheduling
Efficiency: O(n)

C++ Pseudocode:
void roundRobinSignals(vector& lanes, int timeSlice) {
    int n = lanes.size();
    int t = 0;
    while (true) {
        string lane = lanes[t % n];
        cout << "Green for lane: " << lane
             << " for " << timeSlice << " seconds\n";
        t++;
        // continue rotating signals
    }
}

Conclusion:
Round Robin scheduling reduces waiting time, giving each lane fair access to green signals.

Accident Detection

Case Study:
Accidents block roads and delay emergency response. How can anomalies in traffic flow be detected instantly? Can we flag unusual speed drops or density spikes?

Algorithm: Hash Map Anomaly Detection
Efficiency: O(1) average lookup

C++ Pseudocode:
bool detectAccident(vector& speeds) {
    unordered_map freq;
    for (int s : speeds) {
        freq[s]++;
        if (s < 10 && freq[s] > 5) {
            return true; // accident anomaly detected
        }
    }
    return false;
}

Conclusion:
Hash maps quickly detect anomalies, enabling faster accident alerts and emergency response.

Parking Allocation

Case Study:
Parking shortages frustrate drivers in city centers. How can slots be allocated fairly and efficiently? Can we assign nearest available spots using greedy logic?

Algorithm: Greedy Allocation
Efficiency: O(n log n)

C++ Pseudocode:
struct Slot { int distance; bool free; };

void allocateParking(vector& slots) {
    sort(slots.begin(), slots.end(),
         [](Slot a, Slot b){ return a.distance < b.distance; });
    for (Slot& s : slots) {
        if (s.free) {
            s.free = false;
            cout << "Allocated slot at distance: " << s.distance << "\n";
            break;
        }
    }
}

Conclusion:
Greedy allocation ensures drivers get the nearest available slot, reducing search time and congestion.

Emergency Vehicle Routing

Case Study:
Ambulances and fire trucks need fastest routes. How can we guide them through traffic in real time? Can we combine heuristics with shortest path search?

Algorithm: A* Search
Efficiency: O(E log V)

C++ Pseudocode:
struct Node { int id; int g, h; };

vector aStar(int start, int goal,
                  vector>>& adj,
                  vector& heuristic) {
    priority_queue, vector>, greater<>> pq;
    vector dist(adj.size(), INT_MAX);
    dist[start] = 0;
    pq.push({heuristic[start], start});
    while (!pq.empty()) {
        int u = pq.top().second; pq.pop();
        if (u == goal) break;
        for (auto [v, wt] : adj[u]) {
            int gScore = dist[u] + wt;
            int fScore = gScore + heuristic[v];
            if (gScore < dist[v]) {
                dist[v] = gScore;
                pq.push({fScore, v});
            }
        }
    }
    return dist;
}

Conclusion:
A* search guides emergency vehicles along optimal routes, saving critical response time.
